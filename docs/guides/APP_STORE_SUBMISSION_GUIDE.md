## üçé **APPLE APP STORE**

### **1. Create Developer Account**

- Visit: https://developer.apple.com/
- Enroll in Apple Developer Program ($99/year)
- Wait for approval (1-2 days)

### **2. Prepare App**

```bash
cd packages/mobile/shipper
npx expo build:ios
# Repeat for operator and driver
```

### **3. App Store Connect**

1. Create app listings for:
   - Rodistaa Shipper
   - Rodistaa Operator
   - Rodistaa Driver

2. Fill required information:
   - App name
   - Subtitle
   - Description
   - Keywords
   - Support URL
   - Privacy policy URL

3. Upload screenshots (6.5", 5.5")
4. Upload app preview videos (optional)
5. Set pricing (Free)
6. Select countries/regions

### **4. Submit for Review**

- Submit via Xcode or App Store Connect
- Review time: 2-5 days

## ü§ñ **GOOGLE PLAY STORE**

### **1. Create Developer Account**

- Visit: https://play.google.com/console
- Pay one-time $25 fee
- Complete account details

### **2. Prepare App**

```bash
cd packages/mobile/shipper
npx expo build:android
# Repeat for operator and driver
```

### **3. Create Store Listings**

1. Create apps:
   - Rodistaa Shipper
   - Rodistaa Operator
   - Rodistaa Driver

2. Fill required information:
   - App name
   - Short description (80 chars)
   - Full description (4000 chars)
   - Screenshots (phone, tablet)
   - Feature graphic
   - App icon

3. Set content rating
4. Set pricing & distribution

### **4. Submit for Review**

- Upload APK/AAB
- Review time: 1-3 days
