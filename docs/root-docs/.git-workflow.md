# Git-First Workflow Configuration

## Workflow Rules
1. **Always pull from git before making changes**
2. **Commit and push after every action**
3. **Sync git code and documents periodically**
4. **Work only from git repository, not local storage**

## Git Commands Reference

### Before Making Changes
```bash
git fetch origin
git pull origin main
```

### After Making Changes
```bash
git add .
git commit -m "Description of changes"
git push origin main
```

### Periodic Sync
```bash
git fetch origin
git pull origin main
git push origin main
```
