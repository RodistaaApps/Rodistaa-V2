# Franchise SOP: Photo Verification

## Overview

Photo verification tasks are assigned to franchises when trucks have unusual configurations or discrepancies. This SOP ensures consistent, high-quality verification.

## Photo Quality Requirements

- **Resolution**: Minimum 1920x1080 (Full HD)
- **Lighting**: Good daylight or well-lit area
- **Focus**: All photos must be sharp and in focus
- **Angles**: Straight-on views (not angled)
- **Background**: Clear, uncluttered background

## Verification Steps

### Step 1: Review Task
1. Open franchise task list
2. Review truck RC number and assigned flags
3. Note due date (48 hours from assignment)

### Step 2: Locate Truck
1. Verify truck is present at location
2. Confirm RC number matches task
3. Note current condition and accessibility

### Step 3: Take Photos
1. Use checklist above
2. Ensure measuring tape visible in deck length photo
3. Take multiple angles if needed
4. Verify all photos are clear before uploading

### Step 4: Upload Photos
1. Upload all 6 required photos
2. Verify uploads are complete
3. Add notes if any issues encountered

### Step 5: Verification Decision

#### Pass (Verified)
- Photos confirm operator-declared dimensions
- Tyre count matches declaration
- Body length matches declaration (within tolerance)
- Body type matches declaration

#### Fail (Not Verified)
- Dimensions do not match
- Tyre count incorrect
- Body type different
- Photos unclear or incomplete

### Step 6: Submit Result
1. Select Pass or Fail
2. Add notes explaining decision
3. Submit verification

## Common Issues

### Issue: Measuring Tape Not Visible
**Solution**: Retake deck length photo with tape clearly positioned along cargo area

### Issue: Tyre Count Unclear
**Solution**: Take additional close-up photos of tyre axles from different angles

### Issue: Body Type Ambiguous
**Solution**: Take additional photos showing body details (sides, rear, top if accessible)

### Issue: Truck Not Accessible
**Solution**: 
1. Note in task comments
2. Request operator to move truck to accessible location
3. Contact HQ if truck unavailable within SLA

## Escalation

If unable to complete verification:
1. Document reason in task notes
2. Contact HQ support
3. Request task reassignment or extension

- 10% of passed verifications (random sampling)
- 100% of failed verifications
- All verifications with flags raised
