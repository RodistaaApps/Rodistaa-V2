# Mobile App Assets

This directory should contain:

## Required Assets

### icon.png
- Size: 1024x1024px
- Format: PNG with transparency
- Content: Rodistaa logo on transparent background

### splash.png
- Size: 2048x2732px (portrait)
- Format: PNG
- Content: Rodistaa branding with logo

### adaptive-icon.png (Android)
- Size: 1024x1024px
- Format: PNG with transparency
- Content: Rodistaa logo (centered, will be cropped to circle)

### favicon.png (Web)
- Size: 192x192px or 512x512px
- Format: PNG
- Content: Rodistaa logo

## Temporary Placeholders

For development, use:
- Expo default icons (auto-generated)
- Or create simple colored squares with "R" logo

Contact design team for:
